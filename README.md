## Información General
Esta aplicación es un proyecto con el único fin de practicar con tecnologías
de FrontEnd. La aplicación hace uso de la [Api de RIOT](https://developer.riotgames.com)

## Tecnologías
* HTML5
* CSS3
  * Stylus
* Javascript
  * Vuejs
    *Vuetify
* Npm
  * Axios
